Data & Methodology

Where does your data come from?

We aggregate data from multiple reliable sources:

  • Price Data: Binance, Coinbase (real-time spot prices)
  • Market Sentiment: Alternative.me Fear & Greed Index
  • On-Chain Data: Blockchain.info (hashrate, network metrics)
  • Trading Metrics: Binance API (volume, order book data)
  • Market Cap: CoinGecko (dominance, global metrics)

All data is fetched in real-time from public APIs to ensure accuracy.

How does the AI generate signals?

Our analysis uses Claude AI to process multiple data points:

  1. Technical Analysis: RSI, MACD, Bollinger Bands, moving averages, and chart patterns
  2. Market Sentiment: Fear & Greed Index, funding rates, long/short ratios
  3. On-Chain Metrics: Hashrate trends, network activity
  4. Volume Analysis: Buy/sell pressure, liquidity zones

The AI weighs these factors to generate a directional bias (bullish/bearish/neutral) with a confidence score.

Technical & Support

The data isn't loading. What should I do?

Try these steps:

  1. Refresh the page
  2. Check your internet connection
  3. Disable any ad blockers (they may block API calls)
  4. Try a different browser
  5. Clear your browser cache

Some data sources may occasionally be unavailable due to API rate limits or maintenance.

My payment went through but content is still locked?

Lightning payments are instant, but occasionally there may be a brief delay. Try refreshing the page. If the issue persists, your browser may have blocked the local storage where access is tracked. Try disabling strict privacy settings or using a different browser.
