## Interactive API Explorer

Test and explore our Bitcoin market data APIs in real-time. No authentication required for read-only endpoints.

### Available Endpoints

- Market Data - Current BTC price, volume, and market cap
- Fear & Greed Index - Real-time sentiment metrics
- Funding Rates - Perpetual futures funding across exchanges
- Open Interest - Derivatives market positioning
- Liquidations - Recent liquidation data and levels
- On-Chain Metrics - Network fundamentals

### Features

- Live endpoint testing with instant responses
- Automatic code generation (cURL, JavaScript, Python)
- Syntax-highlighted JSON responses
- One-click copy for code examples
- No API key required
